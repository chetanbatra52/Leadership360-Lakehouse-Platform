# Leadership360 Lakehouse Platform

## Overview

Leadership360 Lakehouse Platform is an end-to-end Data Engineering project built using SQL Server, Python and Power BI.

The project simulates an enterprise analytics platform used to process leadership assessment data through a modern multi-layer architecture.
